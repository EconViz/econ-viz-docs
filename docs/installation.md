# Installation

## Requirements

- Python 3.10 or later

## Install from PyPI

```bash
pip install econ-viz
```

## Install for development

```bash
git clone https://github.com/EconViz/econ-viz.git
cd econ-viz
pip install -e ".[dev]"
```

## Verify

```bash
python -c "import importlib.metadata as m; print(m.version('econ-viz'))"
econ-viz help
```
