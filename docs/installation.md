# Installation

## Requirements

- Python 3.12 or later

## Install from PyPI

```bash
pip install econ-viz
```

## Install for development

```bash
git clone https://github.com/EconViz/econ-viz.git
cd econ-viz
pip install -e ".[dev]"
```

## Verify

```bash
python -c "import econ_viz; print(econ_viz.__version__)"
econ-viz --help
```
